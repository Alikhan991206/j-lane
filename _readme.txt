Адаптивная HTML вёрстка из Adobe XD на реальном примере. Проект салон красоты АЙман. 
Проект сделал:Алихан Жаксылыков


Важные примечания к выпуску:
- Наименование папки вашего аккаунта должно быть написано латиницей, например: "C:\Users\alex", а не кириллицей "C:\Users\алекс".
	Это необходимо для корректной работы  Adobe XD.
- Adobe XD разрабатывался для платформы магазина Windows, поэтому если вы пользователь Windows, необходима вервия ОС Windows 10.
